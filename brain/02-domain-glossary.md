# 02 — قاموس المجال (Domain Glossary)

تعريفات موحّدة — مصطلح واحد لكل مفهوم عبر الكود والواجهة والوثائق.

| المصطلح (AR) | Term (EN) | التعريف |
|---|---|---|
| وحدة تنظيمية | Org Unit | عقدة في التسلسل الهرمي للوزارة (قطاع/إدارة/قسم). مالكة للسجلات عبر `OwnerUnitId`. |
| نطاق | Scope | مجموعة الوحدات التنظيمية التي يُسمح للمستخدم برؤيتها/العمل عليها. |
| صلاحية | Permission | إذن ذرّي بصيغة `{module}.{action}` — أصغر وحدة تحكّم بالوصول. |
| دور | Role | حزمة صلاحيات تُسنَد لمستخدم. لا منطق فيها سوى التجميع. |
| موديول | Module | وحدة وظيفية مستقلّة (users, reports…) تُفعَّل/تُضبَط لكل قسم عبر Module Registry. |
| علم تفعيل | Feature Flag | مفتاح في قاعدة البيانات يُفعّل/يُطفئ موديولًا أو ميزة لقسم معيّن. |
| تدقيق | Audit | سجل (من/متى/ماذا/قبل-بعد) لكل كتابة حسّاسة. |
| حذف ناعم | Soft Delete | تعليم السجل محذوفًا (`IsDeleted`) دون إزالته، مع Global Query Filter يُخفيه. |
| نتيجة | Result | غلاف نجاح/فشل (`Result<T>`) بدل الاستثناءات للتحكّم بالتدفّق. |
| معرّف الارتباط | Correlation ID | معرّف يمرّ عبر كل الطبقات لتتبّع طلب واحد في السجلات. |

> أضِف أي مصطلح جديد هنا فور ظهوره في الكود. التناقض في التسمية = دَين تقني.
